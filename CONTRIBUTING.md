# Contributing to Awesome x402

Thank you for your interest in contributing!

## How to Add Your Project

1. Fork this repository
2. Add your project to the relevant section in `README.md`
3. Follow the existing format (name, link, description)
4. Make sure your project:
   - Uses the x402 protocol for payments
   - Has a working demo or live deployment
   - Has documentation available
5. Open a Pull Request with a clear description

## Guidelines

- One project per Pull Request
- Keep descriptions concise (one line)
- Alphabetical order within sections
- No duplicate entries
- Link to the project's main page or repository

## Table Format

For tables, follow the existing column structure. Example:

```markdown
| [Your Project](https://example.com) | Brief description | 5 | USDC | ✅ Live |
```

## Questions?

Open an issue if you have questions about whether your project fits.
